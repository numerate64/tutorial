# GitHub Pages Magic Quest

An interactive, quest-style tutorial site for learning how to publish static websites with GitHub Pages.

## Published Page

```text
https://numerate64.github.io/tutorial/
```

## Files

- `index.html` - tutorial content and section structure.
- `styles.css` - colorful responsive styling and animation rules.
- `script.js` - section navigation, hints, progress interactions, keyboard shortcuts, copy behavior, and easter eggs.
- `_config.yml` - GitHub Pages/Jekyll configuration.

## What It Teaches

- What GitHub Pages is.
- How static hosting works.
- How to create a Pages repository.
- How to commit and push a basic site.
- Practice challenges for styling, navigation, interactivity, and portfolio publishing.

## Local Preview

```sh
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Notes

This is itself a static GitHub Pages site. Progress and interactions are browser-local.
